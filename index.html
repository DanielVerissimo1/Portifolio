Sobre Mim

Daniel Verissimo

Front-End Developer

Desde cedo, sempre fui fascinado por como a tecnologia pode transformar o mundo ao nosso redor. Tenho 17 anos, sou estudante da EEEP Luiz Gonzaga Fonseca Mota e estou cursando o ensino médio integrado ao técnico em Informática. Foi nesse ambiente que descobri o quanto o desenvolvimento web me encanta — e também como a lógica por trás dos sistemas me instiga.

Hoje, me dedico principalmente ao desenvolvimento front-end. Gosto de transformar ideias em interfaces que façam sentido para o usuário — unindo design, usabilidade e desempenho. Além disso, também tenho experiência com banco de dados e design gráfico, o que me permite ter uma visão mais completa de um projeto, do visual à estrutura de dados.

Sou curioso por natureza, sempre explorando novas ferramentas e formas de melhorar meu trabalho. Acredito que a tecnologia é uma ponte entre ideias e soluções, e quero fazer parte dessa construção. Este portfólio é o começo da minha jornada, e fico muito feliz por você estar aqui conhecendo um pouco do que faço.

Tecnologias que eu Trabalho

LINUX

O Linux é meu sistema de desenvolvimento. Ele me dá controle, segurança e produtividade no dia a dia com seus recursos avançados.

HTML

Com HTML, estruturo meus projetos da base ao topo. É o esqueleto das páginas e onde tudo começa a ganhar forma no desenvolvimento web.

CSS

Utilizo o CSS para dar estilo e personalidade às páginas, criando layouts responsivos e experiências visuais bem definidas.

JAVA SCRIPT

Com JavaScript, deixo os sites interativos e dinâmicos. Ele é essencial para funcionalidades modernas e uma navegação fluida.

BOOTSTRAP

Uso Bootstrap para agilizar a criação de interfaces responsivas com um visual limpo, moderno e componentes prontos para usar.

FIGMA

Figma é onde dou vida às ideias visuais. Crio protótipos, layouts e interfaces organizadas para planejar melhor cada projeto
